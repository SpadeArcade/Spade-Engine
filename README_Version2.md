# Spade Engine

**Spade Engine** is a modified Friday Night Funkin' engine built with HaxeFlixel.

## About
This engine is based on the original [Friday Night Funkin'](https://github.com/ninjamuffin99/Funkin) source code, enhanced for new features and multiplayer support.

## Features
- Rhythm gameplay
- Multiplayer support
- Additional custom features

## Getting Started
See the [original FNF documentation](https://github.com/ninjamuffin99/Funkin) for basic usage. This engine extends FNF to provide new functionality.
